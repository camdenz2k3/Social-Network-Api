# Social Network Api
## Description
API for a social network web application where users can share their thoughts, react to friends' thoughts and create a friends list.

It uses Express for routing, a mongoDB database, and the Mongoose ODM
## Usage
The application is invoked by running:
```
npm start
```
after installing the npm packages with
```
npm i
```
